A) HTML Structure

This theme is built in HTML5, and uses the Bootstrap 3.0 Framework. Each page has essentially the same head, as well as navigation and heading (apart from the home page which has a Slider).


Every element between the navigation, and the footer, is contained within a <div> with the class "container".

<div class="container">
<-- Content -->
</div>

Everything must be placed inside .container (fixed width) or .container-fluid (full width) for the Bootstrap Grid system to work properly (This includes the Navigation, and the Footer).

You can read more about grid layouts and how to create rows and columns on the Bootstrap 3.0 Documentation Page at http://getbootstrap.com/css/#grid

B) SASS Files and Structure

This theme is styled in SASS (using .scss files) and includes many, many files that are compiled into one .css file. These .scss files include: Bootstrap components, Bootstrap Variables, CSS Animation, Beaver Slider, IcoMoon, Magnific Popup, and the custom main .scss file.

In order to compile these SASS files, you will first need Ruby and then you will need to install the Compass gem.

You can them simply open the project directory in terminal, and run compass watch. I find this to be the easiest way to compile, since it detects when a change has been made to a .scss file, without having to run a command every time I want to make a change.

Inside the "sass" folder there is a "bootstrap" folder. I have left this untouched, in case you need to update to a newer version of Bootstrap in the future. This folder contains all the bootstrap elements.

Bootstrap Elements Supported in this Theme Include:

  • Dropdowns
  • Tabs
  • Navbar
  • Pagination
  • Pager
  • Code
  • Labels
  • Alerts
  • Progress Bars
  • Accordion/Collapse

Have a look at the Bootstrap 3.0 Docs to learn how to use them.

Inside the "sass" folder there are also several other .scss files.

animate.scss was derived from "animate.css" (simply converted to .scss)

In order to use these animations, simply add the following code to any element.

<div class="animated" data-animation="fadeIn">
<-- Content Here -->
</div>

Replace "fadeIn" with any animations found on the animate.css docs.

beaverslider.scss contains styling for all possible elements for the "Beaver Slider". You can check out these elements at beaverslider.com

As seen on the home page, the slider is added by using one line of code, and can be customized in the beaverslider-settings.js

<div id="top-slider"></div>

icomoon.scss was generated by IcoMoon.io and converted to SASS.

You can use any Icon found on the Icons page.

For example <i class="icomoon-home"></i> generates   

magnific-popup.scss can be found on the magnific popup github page.

It contains all styling for any popups featured in the magnific-settings.js file in the "javascripts" folder.

main.scss includes all the styling for this theme.

It includes custom mixins at the top, followed by theme html elements, headings and classes.

variables.scss includes all the SASS variables for the Theme, Bootstrap, as well as Magnific Popup. This allows for easy changes to styled elements.

Customizing Theme for your own use:

It is very easy to customize Simplifier for your own use. If you simply look in the variables.scss file, you will see all the Bootstrap variables that you can edit, as well as some custom ones that I've added.

For example, under variable options, you will see 

///////////////////
// Theme options //
///////////////////

Currently, the regular "Black Theme" is uncommented. I have provided two alternatives (Navy Theme, and Green Theme), which you can use by commenting out the Black Theme, and uncommenting one of the other themes.

One thing to note, if you look at the bottom of the variables.scss you will see all the SASS settings for Magnific Popup

From Development to Production

When moving your project from development to production, there are some files that will not be neccesary on the web, that are neccesary for SASS.

These files include:

  • .sass-cache (folder)
  • sass (folder)
  • config.rb (file)

Do not include these files in your Production folder

C) Javascript

This theme imports several Javascript Files

Every page in the theme includes the following JS files:

  • jQuery
  • jQuery-Waypoints
  • Bootstrap
  • Main Custom JS File

Extra JS files included on the home page:

  • Beaver Slider
  • Beaver Slider Effects
  • Beaver Slider Settings
  • Magnific Popup
  • Magnific Popup Settings

Every page on the theme does not include these scripts, as they take longer to load, and are not essential on every page. If you wish to use Beaver Slider or Magnific Popup on any pages other than the home page, you can simply add them to the bottom of the body (for example, as done in the Popups page).

 

If you would like to add your own custom JS, you can do so in the main.js file, under the $(document).ready(function(){
