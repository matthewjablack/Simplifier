Responsive 12 Column Grid System Provided by Bootstrap

Bootstrap includes a responsive grid system, which utilizes 12 columns for a 940px wide container. With the responsive CSS file added, the grid adapts to be 724px and 1170px wide depending on your viewport. These grid systems are used for creating page layouts through a series of rows and columns that house your content. To learn more see http://getbootstrap.com/css/#grid
