Image Gallery

Image Gallery Popup

You can use any group of images as an image gallery popup. These images must simply be grouped together in a <div> tag, with the class "popup-gallery", like this <div class="popup-gallery"><--images here--></div>

All you need to do to use these popups, is include the "magnificient-settings.js", and the "magnificient-effects.js" on your page.
